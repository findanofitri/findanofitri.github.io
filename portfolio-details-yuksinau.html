YUKSINAU

Yuksinau is a user interface design for a web application aimed at high school students in preparation for college. Yuksinau has a feature regarding information on campuses in Indonesia. In addition, there is also a feature for the tryout entrance exam and accompanied by a national ranking so that students know the opportunity to get to the destination campus. Creating a Yuksinau User Interface using the figma.com tool. This preparation is done independently for the needs of internship selection.
