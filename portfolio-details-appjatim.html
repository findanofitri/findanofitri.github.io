APP JATIM WEBSITE

The Anti Poverty Program (APP) is one of the policies to accelerate poverty reduction in East Java. The purpose of this program is for the underprivileged to have income and purchasing power through empowering Community Group businesses. The Anti Poverty Program Website project was built using the CodeIgniter Framework and carried out in a team of 5 people.
